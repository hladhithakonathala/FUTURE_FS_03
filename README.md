# 🌸 HDK Boutique Website

## 📌 Project Overview
HD Care Boutique is a modern fashion boutique website developed as part of an internship task.  
It provides an interactive and user-friendly platform for customers to explore fashion collections and place orders online.

---

## 💡 Features

✨ Multi-page Website  
- Home Page  
- Women's Collection  
- Men's Collection  
- Order Page  

🛍️ Shopping Experience  
- Product display with images & prices  
- Size selection (S, M, L, XL)  
- “Buy Now” redirects to order page  

💸 Unique Feature  
- Bargaining option (user can enter their own price)

📦 Order System  
- User can enter name, address, phone number  
- Order confirmation message displayed  

📍 Business Integration  
- Google Maps location  
- Contact details  

🎨 UI/UX Enhancements  
- Image slider (auto moving)  
- Dark mode toggle 🌙  
- Smooth animations  
- Hover effects  
- Loading screen  

⭐ Customer Interaction  
- Feedback form  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- VS Code  
- GitHub  

---

## 🎯 Purpose

This project was created to simulate a real-world business website and demonstrate how local boutiques can build an online presence to attract more customers and improve engagement.

---

## 🚀 Key Learnings

- Building multi-page responsive websites  
- Designing user-friendly UI/UX  
- Adding real-world features like ordering & feedback  
- Implementing JavaScript for interactivity  
- Understanding how business websites function
